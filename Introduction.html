ITPC102 - Object-oriented Programming

This course includes discussion on procedural/functional approach to object-oriented programming approach. It also includes the design and implementation, test and debug programs using OOP concepts such as abstraction, encapsulation, inheritance and polymorphism.

For easier demonstration of examples, this page uses a repl like Trinket. In the Python script on the LEFT, you can type or modify the Python code. If you hit the Run button, your python script (script.py) is executed and the output is shown in the Python Shell on the RIGHT. It also checks whether your code is correct and gives you feedback.

Integrated Development Environment (IDE)

Introduction

Python is a very simple language, and has a very straightforward syntax.

Indentation

Python uses indentation for blocks, instead of curly braces. Both tabs and spaces are supported, but the standard indentation requires standard Python code to use four spaces.
